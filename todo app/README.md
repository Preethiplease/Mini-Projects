# Todo App

A simple Todo application with persistent tasks and a persistent Day/Night theme.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Persist todos using `localStorage`
- Day / Night mode toggle with saved preference

## Concepts Practiced
- DOM updates based on application state
- Arrays and objects
- Event delegation
- CRUD logic (Create, Read, Update, Delete)
- UI state management (completed vs active)
- Clean separation of HTML, CSS, and JavaScript

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
